Scripting and Automation

Analysis, visualization, data management, map automation and production, portal content and organization management tasks can be scripted and automated with Python, the scripting language of choice for the ArcGIS platform.

Sign Up for Free

Why script ArcGIS with Python?

Python is a widely used, free, cross-platform, open-source programming language that is both powerful and easy to learn. Esri has fully embraced Python for ArcGIS and sees Python as the language that fulfills the needs of our user community. Here are just some of the advantages of scripting ArcGIS with Python:

Scalable

Highly scalable, suitable for large projects or small ad-hoc tasks.

Easy to use

Simple and intuitive for beginners, rich coverage for experts.

Capable

Leverage the rich Python ecosystem and integrate ArcGIS with the SciPy stack for scientific computing.

Modern

Utilizes modern techniques and optimized practices to exploit capabilities of the ArcGIS platform.

What are my options for scripting ArcGIS with Python?

The ArcGIS platform provides diverse technology choices for scripting and automation:

  • ArcGIS Python API
  • ArcPy

ArcGIS Python API

ArcGIS Python API is a powerful, modern and easy to use Python SDK for Web GIS.

The SDK consists of
  • Pythonic GIS API
  • python packages delivered via Conda
  • interactive documentation and samples
    • that run both within the Jupyter Notebook environment as well as in a batch processing environment using scripts.
Get ArcGIS Python API Beta

ArcPy

ArcPy is a Python site package that provides a useful and productive way to perform geographic data analysis, data conversion, data management, and map automation with Python.

  • Leverage geoprocessing tools throughout ArcGIS for Desktop.
  • Script with ArcPy to automate powerful geoprocessing functions.
  • Write routines with Python to access all the powerful features in Python.
